Etiquetas en HTML 5

  • figure: Indica una figura ilustrada como parte del documento HTML5.
  • code: con esta etiqueta se verá el texto como si fuera código. Ejemplo: html
  • main: Se usa para definir la parte principal del documento, por lo que solo puede existir uno por documento
  • aside: Dentro de esta suele alojarse el contenido adicional de la web, mismo que suele ser de poca relevancia.
  • footer: Usado para definir el pie de página.
